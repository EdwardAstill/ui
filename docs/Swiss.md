# Swiss

**Theme ID:** `swiss`

A typographic design system rooted in the International Typographic Style (Swiss Style) — the design movement that gave us Helvetica, rigid grid systems, and the belief that form follows function absolutely.

## Visual Character

Pure white background (`#ffffff`) with black text (`#111111`) and a single red accent (`#ff0000`). The layout is governed by a strict invisible grid. No decorative elements, no gradients, no shadows. Visual interest comes entirely from typography: font weight contrasts, size hierarchy, spacing, and the deliberate use of red as the sole color break.

## Typography

- Body/Display: Inter / Helvetica Neue / Helvetica / Arial — as close to the original Swiss typefaces as web fonts allow
- Mono: IBM Plex Mono / Fira Code

## Key Design Traits

- Zero border-radius — grids and geometry are rectilinear
- Red-only accent (`#ff0000`) — no secondary color
- Blue (`#0066ff`) available as `--accent-2` for links and interactive states
- 960px max-width layout column
- Typography-first hierarchy — spacing, weight, and size do all the work
- No box shadows, no gradients, no decorative borders

## Use When

You want to communicate authority, craft, and design literacy. Museums, editorial sites, design portfolios, and brand identity showcases are natural fits. Also useful when you want a layout that looks intentionally designed rather than merely clean.
