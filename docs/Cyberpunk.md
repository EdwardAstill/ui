# Cyberpunk

**Theme ID:** `cyberpunk`

A high-energy, neon-on-black theme inspired by dystopian sci-fi aesthetics. Animated scanlines, a scrolling news ticker, and CRT flicker effects make this the most kinetic theme in the set.

## Visual Character

Pure black background (`#000000`) with neon cyan, magenta, and yellow accents. Heavy use of CSS animations: a vertical scanline sweeps continuously, text occasionally flickers, and a horizontal ticker scrolls news-style content across the bottom. Typography leans into the aesthetic with **Bebas Neue** for display text and **Share Tech Mono** for body — both loaded from Google Fonts.

## Typography

- Display: Bebas Neue (condensed, uppercase)
- Body/Code: Share Tech Mono

## Key Design Traits

- Animated CRT scanline overlay
- Scrolling ticker bar
- Neon glow on key elements via `text-shadow` and `box-shadow`
- Pure black background — maximum contrast
- Flicker animation on containers (subtle, ~3% of frames)
- Zero border-radius — everything is sharp-cornered

## Use When

You want maximum visual drama. Demos, portfolio showpieces, or any context where "wow factor" is more important than calm readability.
