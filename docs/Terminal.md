# Terminal

**Theme ID:** `terminal`

A developer-native dark theme modeled on GitHub's dark color system and modern terminal emulators. Everything is designed to look like you're reading it in a code editor or CLI — monospace by default, syntax-colored status indicators, and a commit-graph style activity chart.

## Visual Character

Deep navy-black background (`#0d1117`) — GitHub's exact dark mode base. Card surfaces step up to `#161b22`. Text is a warm near-white (`#e6edf3`) rather than pure white, reducing eye strain. The accent palette mirrors terminal conventions: blue for info/links, green for success/additions, red for errors/deletions, and yellow for warnings.

## Typography

- Body & Mono: JetBrains Mono / Fira Code / Cascadia Code — monospace everywhere, including body text

## Key Design Traits

- Monospace body font — the entire UI reads like a terminal
- Status color tokens: `--danger` (`#f85149`), `--warning` (`#e3b341`), `--success` (`#39d353`)
- 4px border-radius — minimal rounding, just enough to soften raw edges
- Commit-graph style activity visualization
- Subtle `#30363d` borders — visible but not distracting
- 960px max-width layout

## Use When

You're building for developers, technical users, or anyone who spends their day in a terminal or code editor. CLIs, developer dashboards, code playgrounds, and open-source project sites are the natural home for this theme.
