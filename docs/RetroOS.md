# Cartoon OS

**Theme ID:** `retro-os` — displayed as "Cartoon OS" in the sidebar

A whimsical cartoon-illustrated OS window system. Warm parchment backgrounds, thick dark outlines, hard offset shadows, and pastel title bars give every element the feel of a hand-drawn interface.

## Visual Character

Warm cream/parchment background (`#FAF0DC`) with a subtle dot-grid pattern. All windows (`cwin`) have 3px dark brown borders (`#2D1B00`), rounded corners (10px), and a hard 4px offset shadow — the "cartoon" effect. Title bars come in six named pastel color variants: purple, mint, peach, sky, yellow, and black (with green text). The active sidebar item uses a purple fill with its own hard shadow.

## Typography

- Body: Space Grotesk (weights 400–700)
- Mono/Display: Press Start 2P (pixel font — used for special elements)

## Key Design Traits

- Dot-grid background texture on the page canvas
- Cartoon window chrome (`cwin`, `cwin-titlebar`, `cwin-controls`) with macOS-style close/min/max circles (red/yellow/green) outlined in dark brown
- Six title bar color variants per window
- Hard offset shadow (`4px 4px 0px #2D1B00`) on all windows — no blur
- Dark brown outline (`#2D1B00`) as the universal border color
- Accent: purple (`#6C3FBD`) and coral-red (`#E8543A`)

## Use When

You want something playful and illustrative — a UI that looks like it belongs in a children's app, an indie game, or a quirky creative tool. The cartoon window chrome makes even dry content feel approachable.
