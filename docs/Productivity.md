# Productivity

**Theme ID:** `productivity`

A clean, professional light theme built for focus. Modeled after modern SaaS productivity tools — think Linear, Notion, or Vercel. Everything is optimized for scanability and long working sessions.

## Visual Character

Pure white background with a light gray sidebar (`#F7F8FA`) and very subtle borders (`#E8EAED`). Near-black text (`#1A1A2E`) with a muted gray for secondary content. The primary accent is indigo (`#4F46E5`) with emerald green (`#10B981`) as a confirmational/success color.

## Typography

- Body: Inter (weights 300–700)
- Mono: JetBrains Mono

## Key Design Traits

- 260px fixed sidebar width — wider than default, suited for navigation-heavy apps
- 10px border-radius — slightly rounded without being playful
- Indigo + emerald accent pairing — professional but not cold
- High information density — compact spacing, clear hierarchy
- No decorative elements — every pixel is functional

## Use When

You want a UI that looks like it belongs in a real SaaS product. Dashboards, admin panels, task managers, and developer tools are the natural home for this theme. It stays out of the user's way and lets the content lead.
