# Catppuccin

**Theme ID:** `catppuccin` (Mocha flavor)

A cozy, pastel-on-dark theme based on the popular [Catppuccin](https://catppuccin.com/) color palette. Uses the Mocha variant — the darkest and most saturated of the four Catppuccin flavors.

## Visual Character

Warm dark base (`#1e1e2e`) with a full 14-color accent strip: rosewater, flamingo, pink, mauve, red, maroon, peach, yellow, green, teal, sky, sapphire, blue, and lavender. The palette is deliberately balanced so every color reads clearly against the dark surface without being harsh on the eyes.

## Typography

Relies on the system font stack. The showcase header pairs the flavor name with a small color-strip swatch row showing all 14 accent colors at a glance.

## Key Design Traits

- Dark background with high contrast pastel accents
- No hard borders — soft depth through color differentiation
- Rounded corners, comfortable line heights
- Cat icon (🐱) as the signature brand element
- "Mocha" flavor badge in the header

## Use When

You want a dark theme that feels warm and approachable rather than clinical. Popular in developer tooling, code editors, and terminal configs.
