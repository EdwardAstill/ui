# Y2K

**Theme ID:** `y2k`

A late-90s / early-2000s aesthetic — the era of Flash intros, metallic gradients, glowing text, and "Under Construction" GIFs. Deep purple-black background with neon pink and cyan-green accents, shimmer animations, and spinning decorative elements.

## Visual Character

Near-black deep purple background (`#0a0015`) with a sidebar gradient from dark purple to dark teal. The accent colors are aggressively neon: hot pink (`#ff6bff`) and acid green (`#00ffaa`). Cards use semi-transparent purple (`rgba(20,0,40,0.8)`). The signature effect is a **shimmer animation** — a moving gradient sweep applied to headings and key elements, evoking the metallic text effects that defined the era.

## Typography

- Body: Inter / Space Grotesk
- Mono: Courier New / Lucida Console — the system fonts of the era

## Key Design Traits

- Shimmer animation (`y2k-shimmer`) on display text via `background-clip: text`
- Spinning decorative elements via `y2k-spin` keyframes
- Deep purple/black with neon pink+green — the Y2K signature palette
- Semi-transparent cards — translucency was a key 2000s motif
- 8px border-radius — slightly more rounded than the brutalist themes
- CSS animations are the primary visual interest, not static styling

## Use When

You want nostalgia for the internet's maximalist, chaotic early years. Retro-web projects, tongue-in-cheek "under construction" pages, throwback branding, or any context where leaning into the aesthetic absurdity of that era is the point.
