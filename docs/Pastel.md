# Pastel

**Theme ID:** `pastel`

A soft, airy light theme built around a lavender/purple palette with gentle mint, peach, rose, and sky accent colors. Rounded and approachable — the friendliest theme in the set.

## Visual Character

Near-white lavender background (`#fdf6ff`) with a slightly tinted sidebar (`#f5eeff`). Cards are white with a soft purple border (`#e5d5f5`). All text is a muted plum (`#3d2952`) rather than harsh black. The accent palette extends into six named colors: mint, peach, rose, sky, lavender, and pink — available as CSS custom properties.

## Typography

- Body: DM Sans (optical size-aware, weights 300–700)
- Mono: JetBrains Mono / Fira Code

## Key Design Traits

- Large border-radius: `16px`
- No hard shadows — elevation via subtle border color
- Six extended palette tokens: `--mint`, `--peach`, `--rose`, `--sky`, `--lavender`, `--pink`
- Muted plum text instead of black — reduces harshness
- Italic variant of DM Sans used in select places for warmth

## Use When

You want a light theme that feels gentle, feminine, or playful. Good for lifestyle apps, creative tools, children's products, or any design that should feel inviting rather than corporate.
