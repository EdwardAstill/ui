# Complete Theme Definitions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use subagent-driven-development (recommended) or executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Complete the theme definitions for 20 themes in `src/themes/` and ensure their `Showcase` components correctly handle `colors` and `layout` props.

**Architecture:** Each theme file is self-contained. The `Showcase` component in each file must be updated to apply the `colors` prop as CSS variables on its root element. The `export const XXXTheme` object must be populated with all required `ThemeDefinition` fields.

**Tech Stack:** React, TypeScript, CSS Variables.
